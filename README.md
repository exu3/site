# Personal Website V2

This website is built with Next.js and Tailwind CSS.

How to run it locally:

```shell
git clone https://github.com/exu3/v2 && cd v2
```

Install dependencies with npm
```sh
npm install
```

Then

```
npm run dev
```

Visit http://localhost:3000 to see the result.
